ASF Project Birthdays, April 2026

Published on Wednesday, Apr 1, 2026 by Rich Bowen

The following Apache projects are celebrating their birthdays in April.

Apache DataFusion (2 years) ¶

Established: 2024-04

Description: The mission of Apache DataFusion is the creation and maintenance of software related to an extensible query engine

Project Homepage: https://datafusion.apache.org

Apache SDAP (2 years) ¶

Established: 2024-04

Description: The mission of Apache SDAP is the creation and maintenance of software related to an integrated data analytic center for Big Science problems

Project Homepage: https://sdap.apache.org

Apache ShardingSphere (6 years) ¶

Established: 2020-04

Description: The mission of Apache ShardingSphere is the creation and maintenance of software related to a database clustering system providing data sharding, distributed transactions, and distributed database management

Project Homepage: https://shardingsphere.apache.org

Apache NetBeans (7 years) ¶

Established: 2019-04

Description: The mission of Apache NetBeans is the creation and maintenance of software related to development environment, tooling platform, and application framework

Project Homepage: https://netbeans.apache.org/

Apache PLC4X (7 years) ¶

Established: 2019-04

Description: The mission of the Apache PLC4X project is creating a set of libraries for communicating with industrial programmable logic controllers (PLCs) using a variety of protocols but with a shared API.

Project Homepage: https://plc4x.apache.org/

Apache SkyWalking (7 years) ¶

Established: 2019-04

Description: The mission of Apache SkyWalking is the creation and maintenance of software related to application performance management and monitoring (APM)

Project Homepage: https://skywalking.apache.org/

Apache CarbonData (9 years) ¶

Established: 2017-04

Description: The mission of Apache CarbonData is the creation and maintenance of software related to an indexed columnar data format for fast analytics on big data platforms

Project Homepage: https://carbondata.apache.org/

Apache Fineract (9 years) ¶

Established: 2017-04

Description: The mission of Apache Fineract is the creation and maintenance of software related to a core banking platform that provides a reliable, robust, and affordable solution for entrepreneurs, financial institutions, and service providers to offer financial services to the world’s underbanked and unbanked.

Project Homepage: https://fineract.apache.org/

Apache AsterixDB (10 years) ¶

Established: 2016-04

Description: The mission of AsterixDB is the creation and maintenance of software related to open source Big Data Management System

Project Homepage: https://asterixdb.apache.org/

Apache Johnzon (10 years) ¶

Established: 2016-04

Description: The mission of Johnzon is the creation and maintenance of software related to JSR-353 compliant JSON parsing; modules to help with JSR-353 as well as JSR-374 and JSR-367

Project Homepage: https://johnzon.apache.org/

Apache ORC (11 years) ¶

Established: 2015-04

Description: The mission of ORC is the creation and maintenance of software related to the smallest, fastest columnar storage for Hadoop workloads

Project Homepage: https://orc.apache.org/

Apache Parquet (11 years) ¶

Established: 2015-04

Description: The mission of Parquet is the creation and maintenance of software related to columnar storage format available to any project in the Apache Hadoop ecosystem

Project Homepage: https://parquet.apache.org/

Apache DeltaSpike (13 years) ¶

Established: 2013-04

Description: The mission of DeltaSpike is the creation and maintenance of software related to Portable CDI extensions that provide useful features for Java application developers

Project Homepage: https://deltaspike.apache.org/

Apache Creadur (14 years) ¶

Established: 2012-04

Description: The mission of Creadur is the creation and maintenance of software related to Comprehension and auditing of software distributions

Project Homepage: https://creadur.apache.org/

Apache Jena (14 years) ¶

Established: 2012-04

Description: The mission of Jena is the creation and maintenance of software related to Java framework for building Semantic Web applications

Project Homepage: https://jena.apache.org/

Apache Avro (16 years) ¶

Established: 2010-04

Description: The mission of Avro is the creation and maintenance of software related to A Serialization System

Project Homepage: https://avro.apache.org/

Apache HBase (16 years) ¶

Established: 2010-04

Description: The mission of HBase is the creation and maintenance of software related to Apache Hadoop Database

Project Homepage: https://hbase.apache.org/

Apache Mahout (16 years) ¶

Established: 2010-04

Description: The mission of Mahout is the creation and maintenance of software related to Scalable machine learning library

Project Homepage: https://mahout.apache.org/

Apache Nutch (16 years) ¶

Established: 2010-04

Description: The mission of Nutch is the creation and maintenance of software related to Open Source Web Search Software

Project Homepage: https://nutch.apache.org/

Apache Tika (16 years) ¶

Established: 2010-04

Description: The mission of Tika is the creation and maintenance of software related to Content Analysis and Detection Toolkit

Project Homepage: https://tika.apache.org/

Apache Traffic Server (16 years) ¶

Established: 2010-04

Description: The mission of Traffic Server is the creation and maintenance of software related to A fast, scalable and extensible HTTP/1.1 compliant caching proxy server

Project Homepage: https://trafficserver.apache.org/

Apache CXF (18 years) ¶

Established: 2008-04

Description: The mission of CXF is the creation and maintenance of software related to Service Framework

Project Homepage: https://cxf.apache.org/
