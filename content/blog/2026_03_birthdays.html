ASF Project Birthdays, March 2026

Published on Sunday, Mar 1, 2026 by Rich Bowen

The following Apache projects are celebrating their birthdays in March.

Apache Celeborn (2 years) ¶

Established: 2024-03

Description: The mission of Apache Celeborn is the creation and maintenance of software related to an intermediate data service for big data computing engines to boost performance, stability, and flexibility

Project Homepage: https://celeborn.apache.org

Apache Paimon (2 years) ¶

Established: 2024-03

Description: The mission of Apache Paimon is the creation and maintenance of software related to a unified lake storage to build dynamic tables for both stream and batch processing with big data compute engines, supporting high-speed data ingestion and real-time data query

Project Homepage: https://paimon.apache.org

Apache Pekko (2 years) ¶

Established: 2024-03

Description: The mission of Apache Pekko is the creation and maintenance of software related to a toolkit and an ecosystem for building highly concurrent, distributed, reactive and resilient applications for Java and Scala

Project Homepage: https://pekko.apache.org

Apache EventMesh (3 years) ¶

Established: 2023-03

Description: The mission of Apache EventMesh is the creation and maintenance of software related to a fully serverless platform used to build distributed event-driven applications

Project Homepage: https://eventmesh.apache.org

Apache Flagon (3 years) ¶

Established: 2023-03

Description: The mission of Apache Flagon is the creation and maintenance of software related to thin-client behavioral logging capability useful for business analytics, usage analytics, usability and user testing

Project Homepage: https://flagon.apache.org

Apache YuniKorn (4 years) ¶

Established: 2022-03

Description: The mission of Apache YuniKorn is the creation and maintenance of software related to a standalone resource scheduler responsible for scheduling batch jobs and long-running services on large scale distributed systems running in on-premises environments as well as different public clouds

Project Homepage: https://yunikorn.apache.org

Apache DolphinScheduler (5 years) ¶

Established: 2021-03

Description: The mission of Apache DolphinScheduler is the creation and maintenance of software related to a distributed and extensible workflow scheduler platform with powerful DAG visual interfaces

Project Homepage: https://dolphinscheduler.apache.org

Apache FreeMarker (8 years) ¶

Established: 2018-03

Description: The mission of Apache FreeMarker is the creation and maintenance of software related to a template engine.

Project Homepage: https://freemarker.apache.org/

Apache Allura (12 years) ¶

Established: 2014-03

Description: The mission of Apache Allura is the creation and maintenance of software related to a software development infrastructure platform commonly known as a “forge”

Project Homepage: https://allura.apache.org/

Apache CloudStack (13 years) ¶

Established: 2013-03

Description: The mission of CloudStack is the creation and maintenance of software related to Infrastructure as a Service solution

Project Homepage: https://cloudstack.apache.org/

Apache cTAKES (13 years) ¶

Established: 2013-03

Description: The mission of cTAKES is the creation and maintenance of software related to Natural language processing (NLP) tool for information extraction from electronic medical record clinical free-text

Project Homepage: https://ctakes.apache.org/

Apache Accumulo (14 years) ¶

Established: 2012-03

Description: The mission of Apache Accumulo is the creation and maintenance of software related to a robust, scalable, distributed key/value store with cell-based access control and customizable server-side processing.

Project Homepage: https://accumulo.apache.org/

Apache UIMA (16 years) ¶

Established: 2010-03

Description: Apache UIMA () software provides frameworks, tools and annotators, facilitating the analysis of unstructured content such as text, audio and video. () Unstructured Information Management Architecture.

Project Homepage: https://uima.apache.org/

Apache Felix (19 years) ¶

Established: 2007-03

Description: The mission of Felix is the creation and maintenance of software related to OSGi Framework and components.

Project Homepage: https://felix.apache.org/

Apache Jackrabbit (20 years) ¶

Established: 2006-03

Description: The mission of Jackrabbit is the creation and maintenance of software related to Content Repository for Java

Project Homepage: https://jackrabbit.apache.org/

Apache Struts (22 years) ¶

Established: 2004-03

Description: The mission of Struts is the creation and maintenance of software related to Model 2 framework for building Java web applications

Project Homepage: https://struts.apache.org/

Apache Maven (23 years) ¶

Established: 2003-03

Description: The mission of Maven is the creation and maintenance of software related to Java project management and comprehension tools

Project Homepage: https://maven.apache.org/

Apache mod_perl (26 years) ¶

Established: 2000-03

Description: The mission of Perl is the creation and maintenance of software related to Dynamic websites using Perl

Project Homepage: https://perl.apache.org/
