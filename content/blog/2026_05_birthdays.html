ASF Project Birthdays, May 2026

Published on Friday, May 1, 2026 by Rich Bowen

The following Apache projects are celebrating their birthdays in May.

Apache Gravitino (1 year) ¶

Established: 2025-05

Description: The mission of Apache Gravitino is the creation and maintenance of software related to managing your data and AI assets seamlessly with a flexible, unified governance framework, including lakehouse federation capabilities

Project Homepage: https://gravitino.apache.org

Apache StormCrawler (1 year) ¶

Established: 2025-05

Description: The mission of Apache StormCrawler is the creation and maintenance of software related to an open source collection of resources for building low-latency, scalable web crawlers on Apache Storm

Project Homepage: https://stormcrawler.apache.org

Apache SeaTunnel (3 years) ¶

Established: 2023-05

Description: The mission of Apache SeaTunnel is the creation and maintenance of software related to a very easy-to-use ultra-high-performance distributed data integration platform that supports real-time synchronization of massive data

Project Homepage: https://seatunnel.apache.org

Apache AGE (4 years) ¶

Established: 2022-05

Description: The mission of Apache AGE is the creation and maintenance of software related to a multi-model database that enables graph and relational models built on PostgreSQL

Project Homepage: https://age.apache.org

Apache Hudi (6 years) ¶

Established: 2020-05

Description: The mission of Apache Hudi is the creation and maintenance of software related to providing atomic upserts and incremental data streams on Big Data

Project Homepage: https://hudi.apache.org

Apache Iceberg (6 years) ¶

Established: 2020-05

Description: The mission of Apache Iceberg is the creation and maintenance of software related to managing huge analytic datasets using a standard at-rest table format that is designed for high performance and ease of use

Project Homepage: https://iceberg.apache.org

Apache Dubbo (7 years) ¶

Established: 2019-05

Description: The mission of Apache Dubbo is the creation and maintenance of software related to a high-performance, lightweight, java based RPC framework.

Project Homepage: https://dubbo.apache.org/

Apache SystemDS (9 years) ¶

Established: 2017-05

Description: The mission of Apache SystemDS is the creation and maintenance of software related to declarative, large-scale machine learning that compiles to hybrid runtime execution plans ranging from single node, in-memory computations, to distributed computations such as on Apache Hadoop MapReduce or Apache Spark.

Project Homepage: https://systemds.apache.org/

Apache TinkerPop (10 years) ¶

Established: 2016-05

Description: The mission of TinkerPop is the creation and maintenance of software related to A graph computing framework for both graph databases (OLTP) and graph analytic systems (OLAP)

Project Homepage: https://tinkerpop.apache.org/

Apache Zeppelin (10 years) ¶

Established: 2016-05

Description: The mission of Zeppelin is the creation and maintenance of software related to A web-based notebook that enables interactive data analytics

Project Homepage: https://zeppelin.apache.org/

Apache Whimsy (11 years) ¶

Established: 2015-05

Description: The mission of Apache Whimsy is the creation and maintenance of software related to tools that help automate various administrative tasks or information lookup activities

Project Homepage: https://whimsical.apache.org/

Apache Phoenix (12 years) ¶

Established: 2014-05

Description: The mission of Phoenix is the creation and maintenance of software related to High performance relational database layer over Apache HBase for low latency applications

Project Homepage: https://phoenix.apache.org/

Apache ManifoldCF (14 years) ¶

Established: 2012-05

Description: The mission of ManifoldCF is the creation and maintenance of software related to Framework for connecting source content repositories to target repositories or indexes.

Project Homepage: https://manifoldcf.apache.org/

Apache Libcloud (15 years) ¶

Established: 2011-05

Description: The mission of Libcloud is the creation and maintenance of software related to Unified interface to the cloud

Project Homepage: https://libcloud.apache.org/

Apache OpenJPA (19 years) ¶

Established: 2007-05

Description: The mission of OpenJPA is the creation and maintenance of software related to OpenJPA: Object Relational Mapping for Java

Project Homepage: https://openjpa.apache.org/

Apache POI (19 years) ¶

Established: 2007-05

Description: The mission of POI is the creation and maintenance of software related to Java API for OLE 2 Compound and OOXML Documents

Project Homepage: https://poi.apache.org/

Apache TomEE (19 years) ¶

Established: 2007-05

Description: The mission of TomEE is the creation and maintenance of software related to Java EE Web Profile built on Apache Tomcat

Project Homepage: https://tomee.apache.org/

Apache Turbine (19 years) ¶

Established: 2007-05

Description: The mission of Turbine is the creation and maintenance of software related to A Java Servlet Web Application Framework and associated component library

Project Homepage: https://turbine.apache.org/

Apache Tomcat (21 years) ¶

Established: 2005-05

Description: The mission of Tomcat is the creation and maintenance of software related to A Java Servlet and JSP Container

Project Homepage: https://tomcat.apache.org/

Apache Geronimo (22 years) ¶

Established: 2004-05

Description: The mission of Geronimo is the creation and maintenance of software related to Java2, Enterprise Edition (J2EE) container

Project Homepage: https://geronimo.apache.org/
