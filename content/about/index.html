Tools and Code we Maintain ¶

ComDev manages a number of helpful tools for finding your way around.

  • The Apache Projects Directory lists all our software projects. See the Project Directory code and the JSON data feeds available.
  • Apache Committers Phone Book lists all individual Apache committers. See the Phonebook code.
  • The Project Reporting Tool (requires login) helps Apache PMCs create quarterly board reports. See the Reporter code.
  • The ComDev Wiki is also available for scratch or experimental work, although most permanent content should be here in the website.

The ComDev PMC maintains these tools, so bring questions to our dev@ list.

About This Website ¶

The Apache Community Development PMC maintains this community.apache.org website and works through the dev@community.apache.org archived mailing list.

See our Newcomer FAQ for how to suggest changes to this website.

The content is written in Markdown and published automatically via a Jenkins job. Source code is hosted in a Git repository.

See the README.md for more info.
